Roei's Project 2 - Crypto Tracking Site

About the project

I'm a passionate full-stack web developer learning to build responsive, dynamic, and user-friendly applications. This project demonstrates my ability to integrate modern front-end techniques, asynchronous programming, and real-time data visualization into a cohesive, professional platform.

This project is a full-stack web application built with a strong emphasis on data visualization, responsive design, and user experience.

The goal of this project was to create an interactive crypto-tracking platform that fetches live data from public APIs and dynamically renders it in a user-friendly, visually engaging interface.

The project utilizes HTML5, CSS3, and Bootstrap for structural layout and styling, ensuring responsiveness and a clean, modern UI. Interactivity and dynamic behavior are driven by Vanilla JavaScript, without reliance on frameworks, showcasing direct DOM manipulation, component rendering, and state management.

Real-time pricing data is fetched using the Fetch API with asynchronous programming patterns, including async/await, setInterval, and error handling. API endpoints from CoinGecko and CryptoCompare are integrated to retrieve both static metadata and live pricing updates.

The application uses Chart.js to present real-time updates in an animated line chart, with data updated every second. User selections are preserved using localStorage, and coin selection is limited to 5, with an elegant modal interface for managing tracked symbols.

Additional techniques include dynamic HTML generation (DHTML), custom loader animations, parallax backgrounds, responsive grid layouts, and modular code separation (utils.js, reports.js, etc.).

This project demonstrates a complete front-end solution for integrating external data sources, handling state and UI logic, managing local persistence, and delivering a polished, production-ready interface - all built with native web technologies.
